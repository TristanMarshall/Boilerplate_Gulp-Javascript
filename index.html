Gulp is awesome

Lets bundle and minify stuff!

  1. to get started with gulp, I need to create a gulpfile.js
  2. In our gulp file, we need to require a few libraries.
  3. gulp, gulp-uglify, gulp.concat, gulp-watch
  4. Gulp is for running tasks
  5. Uglify is for minification/obfuscation
  6. Concat is for concatenation
  7. Watch will listen for changes to your code and auto-run the tasks when changes are made
  8. We ten need to define settings, check the gulpfile for an example
  9. After that, I create my first gulp.task name default
  10. Gulp requires that there is a default taks
